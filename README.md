#  HR Analytics Dashboard (Excel)

##  Project Overview

This project showcases an HR Analytics Dashboard built using Microsoft Excel to analyze employee data and provide meaningful insights for workforce management. The dashboard leverages pivot tables and visualizations to present key HR metrics in an interactive format.

## Objective

To analyze employee data and help organizations:

* Monitor workforce distribution
* Evaluate employee performance
* Understand salary trends
* Support data-driven HR decision

## Key Insights

* Total number of employees across departments
* Average salary (INR) by department (HR, Operations, Marketing, Sales, R&D)
* Employee performance rating distribution
* Department-wise comparison of workforce metrics
* Identification of trends in employee data

## Tools & Techniques Used

* Microsoft Excel (.xlsm)
* Pivot Tables
* Data Aggregation
* Dashboard Design & Visualization

## Dashboard Features

* Interactive pivot-based dashboard
* Department-level analysis
* KPI indicators (Employee Count, Average Salary)
* Clean and structured layout for easy interpretation

##  Business Impact

This dashboard helps HR teams to:

* Improve workforce planning
* Analyze departmental performance
* Make informed salary decisions
* Enhance overall HR efficiency
